# ConsoleLoggerとは....
Proxyを使用しConsoleとサーバーの間で行われている通信を監視する。
# 使用方法
1. CosoleLoggerでportを入力(4桁)

2. PS4のネットワーク設定でカスタムを選択し 1.で入力した4桁をポートに入力する。


4. ネットワーク接続を確認する。(ConsoleLoggerのLogに色々と表示された成功 表示されない場合はportとIPv4を確認)
